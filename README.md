# Описание
Обычный бот с специфичным функционалом.

Я его создал, что бы научится писать код и работать с гитхабом.

Не воспринимайте проект в серьез.

## Запуск бота.
### Запуск бота для Linux
```sh
export BOT_TOKEN=token
python3 main.py
```
> [!IMPORTANT]
> token = Путь до вашего токена бота. Не сам токен.
> 
> В линуке можно создать демон для автоматического запуска бота.

### Запуск бота для Windows
- Установите Питон.
- В настройках переменных среды, указать путь до токена бота. Переменную назвать BOT_TOKEN.
- Запускайте бота с помощью main.py
- В планировщике задач или в сервисах можно настроить автозапуск бота. Гуглите.
